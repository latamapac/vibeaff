# VibeAff Roadmap: MVP to Production

## Phase 0: MVP Delivery
- Core tracking and attribution with verified conversions.
- Affiliate onboarding, campaign creation, and payout queue.
- Basic creatives workflow and channel selection.
- Initial backoffice dashboard with payout holds.

## Phase 1: MVP Hardening
- Reliability: retries, idempotency, and alerting.
- Observability: metrics, logs, traces, and dashboards.
- Audit logs for key actions (payouts, approvals, program edits).
- Support tooling: ticketing integration and admin notes.

## Phase 2: Scale + Compliance
- Multi-tenant isolation and data partitioning.
- SOC2-ready controls and access reviews.
- KYC/AML for crypto payouts and higher-risk affiliates.
- Automated payout compliance checks.

## Phase 3: Growth + Optimization
- Multi-touch attribution models and cross-device tracking.
- ML-based fraud scoring and auto-hold workflows.
- Automated creative testing and performance recommendations.
- Expanded channel integrations and dynamic budget optimization.
